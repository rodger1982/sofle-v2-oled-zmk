# Rodger's Sofle

Custom ZMK config for Rodger's wireless Sofle OLED keyboard.

Features:
- Bluetooth name: `Rodger's Sofle`
- OLED enabled using normal Sofle display support
- Encoder support
- RGB underglow enabled
- 6 layers: `base`, `nav`, `system`, `fn`, `gaming`, `mouse`
- Gaming layer is toggled with `&tog GAMING`
- Mouse layer is momentary with `&mo MOUSE`

Notes:
- The OLED will show ZMK's normal status screen/layer status.
- A true custom boot logo needs custom ZMK display source code and is not included in this safe build package.
- RGB controls are on the `fn` layer. Fully automatic per-layer RGB needs custom ZMK behavior/module work and is not included in this safe build package.
